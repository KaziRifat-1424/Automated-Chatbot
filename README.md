🚀 How It Works

The chatbot follows a simple workflow:

Takes user input from the console
Processes the input using predefined logic or conditions
Matches input with responses
Returns the appropriate reply

Chatbots like this are typically built using pattern matching or predefined responses before moving to advanced AI-based models .
